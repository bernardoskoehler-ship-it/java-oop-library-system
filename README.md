# 📚 Java OOP Library System

A simple library management system built in Java to practice Object-Oriented Programming concepts.

---

## 🚀 About the Project

This project simulates a basic library system where books can be borrowed and returned.

It was created as part of my Java learning journey, focusing on building real logic instead of only solving small exercises.

---

## 🧠 Concepts Practiced

* Classes and Objects
* Encapsulation
* Methods and validation
* Arrays and basic data handling
* Simple business logic

---

## ⚙️ Features

* List all books
* Check if a book is available
* Borrow a book
* Return a book
* Track who borrowed the book

---

## 🏗️ Project Structure

* `Livro` → Represents a book
* `Biblioteca` → Manages the list of books
* `AlugarDevolver` → Interface for borrow/return actions

---

## 💡 Purpose

This project is part of my progression from basic exercises to real-world logic implementation in Java.
